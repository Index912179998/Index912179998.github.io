JAVA面向对象

 Java是面向对象的程序设计语言，Java语言提供了定义类、成员变量、方法等的基本功能。
 面向对象是一个抽象的概念，它是相对于面向过程而言的，过程和对象都是一种解决问题的思想，也是种很人性化的编程思想。
 面向过程强调的是功能行为，一种过程，先做什么，然后再做什么。
 面向对象强调的是将功能封装到对象里，重点是具备某种功能的对象。

1.什么是类和对象?

 面向对象的程序设计过程中有两个重要概念：类(class)和对象(object,也称作实例instance),其中类是某一批对象的抽象，可以把类理解成某种概念；对象才是一个具体存在的实体，从这个意义上来看，日常所说的人，其实都是人的实例，而不是人类。

例子：

public class Computer{
    //定义了两个计算机所拥有的属性也称字段
    public String type;
    public String madedate;
    //这里用了空的构造方法(构造器)
    public Computer(){}
    //定义了计算机桌面的方法
    public void Windows(){
        System.out.println("Welcome!");
    }
}

2.构造器

 特点：方法名与类名相同，不用定义返回值类型 ，不需要写return语句
 作用：给类中的字段进行初始化，可以用来创建对象。
 注意：默认构造方法的特点。，多个构造方法是以重载的形式存在的

3.常用关键字

  static关键字：
  特点：随着类的加载而加载，优先于对象存在，被所有对象所共享，可以直接被类名调用
  使用注意：静态方法只能访问静态成员，但是非静态成员可以访问静态成员;静态方法中不可以使用this，super关键字
  this关键字：
  特点：this表示当前对象。,当前对象 ←→ 当前正在调用实例成员的对象,也就是说谁调用了方法，谁就是当前对象。
  import关键字：
  作用：访问带有package的类，导入类，更好的翻译是“寻找”类：import java.util.Arrays；寻找某个包下被使用到的全部类：import java.util.*;

4.访问修饰符

 private：类内部可以访问
 default：类内部、同包内可以访问
 protected：类内部、同包内、子类可以访问
 public：任何地方可以访问

面向对象的三个特征：

 封装(Encapsulation);
  1.把对象的状态和行为看成一个统一的整体，将二者存放在一个独立的模块中(类)；
  2.信息隐藏，把不需要让外界知道的信息隐藏起来,尽可能隐藏对象功能实现细节,字段;

 继承(Inheritance);
  将不同子类中的共性抽象成父类,在原有父类的基础上加入一些个性,就是子类继承父类，让类与类之间产生关系,是多态的前提。

 多态(Polymorphism);
  编译时的类型由声明该变量时使用的类型决定,运行时的类型由实际赋给变量的对象决定。如果编译时类型和运行时类型不同，就出现多态，总的来说就是指同一个实体同时具有多种形式
