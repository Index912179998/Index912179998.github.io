Java常用类整理

String类

 String表示字符串,所谓字符串,就是一连串的字符;
 String是不可变类，一旦String对象被创建，包含在对象中的字符序列(内容)是不可变的，直到对象被销毁，因为String使用private final char value[]来实现字符串的存储,也就是说String对象创建之后,就不能再修改此对象中存储的字符串内容,就是因为如此,才说String类型是不可变的。

1.String(StringBuffer buffer):根据StringBuffer对象来创建String对象；
2.String(StringBuilder builder):同上

StringBuffer sb = new StringBuffer();
sb.append("想要追加的字符串")；//可以用append方法在字符串后面追加内容。

问题：stringbuffer stringbuilder区别？
答：StringBuilder类不是线程安全的，但其在单线程中的性能比StringBuffer高。

3.char charAt(int index):取字符串中指定位置的字符，index从0开始计算。

String a ="abcde";
System.out.println(a.charAt(2));//输出“c”

4.String concat(String str):连接字符串，等同于 “+”；

5.boolean equals(Object obj):将该字符串与指定对象比较，若二者包含的字符序列相等返回true；

if("abc"equals("ABC"))//结果为0

6.boolean equalsIgnoreCase(String anotherString) 将此 String 与另一个 String 比较，不考虑大小写;

if("abc"equals("ABC"))//结果为1

7.int indexOf(String str):找出str字符串在该字符串中第一次出现的位置；
8.int length():返回当前字符串长度；
9.String replace(char oldChar, char newChar) :返回一个新的字符串，它是通过用 newChar 替换此字符串中出现的所有 oldChar 得到的。
10.String[] split(String regex)： 把字符串按指定的字符串分隔开。
11.String trim() ：忽略字符串前导空白和尾部空白。
12.String substring(int beginIndex, int endIndex) 返回一个新字符串，它是此字符串的一个子字符串;[begin,end)
13.char[] toCharArray() 将此字符串转换为一个新的字符数组。
14.String toLowerCase() 将此 String 中的所有字符都转换为小写;
15.String toUpperCase()转成大写;

Math类

1.abs(X x):求绝对值
2.max(X x1,X x2):求最大值
3.min(X x1,X x2)：求最小值

Random类

1.Random() 创建一个新的随机数生成器。
2.int nextInt() 返回下一个伪随机数，它是此随机数生成器的序列中均匀分布的 int 值。
3.int nextInt(int n) 返回一个伪随机数，它是取自此随机数生成器序列的、在 0（包括）和指定值n（不包括）之间均匀分布的 int 值。 

//在A B C D E F 六人中随机抽取3人中奖，要求中奖人不能重复。
public class MyTest
{
    public static void main(String[] args)
    {
        Vector a = new Vector();
        for(char i='A'; i<='F'; i++) 
              a.add("" + i);   
        for(int k=0; k<3; k++)
        {
            int d = (new Random()).nextInt(a.size());
            System.out.println(a.remove(d));
        }
    }
}

String和Date的相互转换

public static Date string2Date(String str, String pattern) {
    SimpleDateFormat sdf = new SimpleDateFormat(pattern);
    return sdf.parse(str);
}


public static String date2String(Date date, String pattern)
{
    if (date != null) {
        return new SimpleDateFormat(pattern).format(date);
    }
        return "";
