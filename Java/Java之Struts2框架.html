Java之Struts框架笔记

 Struts2 是一个用来开发 MVC应用程序的框架. 是实现SSH的一个点。它提供了 Web 应用程序开发过程中的一些常见问题的解决方案:

1.Struts2 = Struts1 + WebWork 1和2没有本质的关系
2.对页面导航活动进行管理
3.对来自用户的输入数据进行合法性验证
4.统一的布局
5.可扩展性
6.国际化和本地化
7.支持 Ajax
8.表单的重复提交

Struts学习要点：

 常量、路径
 类型转换
 上传与下载
 拦截器
 ognl
 数据校验
 国际化
 模型驱动、防重复提交、数据回显

搭建Struts2的基本步骤

1.创建web应用项目
2.下载struts jar包
3.导入jar包

4.在web.xml当中加入

5.写src/struts.xml-->编译之后-->classes/struts.xml

<?xml version="1.0" encoding="UTF-8"?>
<!DOCTYPE struts PUBLIC
        "-//Apache Software Foundation//DTD Struts Configuration 2.3//EN"
        "http://struts.apache.org/dtds/struts-2.3.dtd">

<struts>

    <constant name="struts.devMode" value="true" />

    <package name="test"  extends="struts-default">
        <!--  http://localhsot:8080/index.action -->
        <action name="index">
            <result>/index.jsp</result>
        </action>
        <!--  http://localhsot:8080/hello.action -->
        <action name="hello" class="com.dazhi.controller.HelloWorldAction" method="hello">
            <result name="songjun">/songjun.jsp</result>
            <result name="success">/hello.jsp</result>
        </action>
        <action name="world" class="com.dazhi.controller.HelloWorldAction" method="world">
            <result name="world">/world.jsp</result>
            <result name="success">/hello.jsp</result>
        </action>
        <!--URL 通配-->
        <!--http://localhost:8080/hw_hello.action-->
        <!--http://localhost:8080/hw_world.action-->
        <!--http://localhost:8080/hw_sdfsfd.action ✖-->
        <action name="hw_*_*" method="{1}" class="com.dazhi.controller.HelloWorldAction" >
            <result name="world" type="redirect">/world.jsp</result>
            <result name="success">/hello.jsp</result>
            <result name="songjun">/songjun.jsp</result>
        </action>
    </package>

</struts>

6:访问http://localhost:8080/index.action
